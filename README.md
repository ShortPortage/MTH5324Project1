# World Bank Development Indicators Analysis

This project analyzes World Bank Development Indicators to model Gross National Income (GNI) per capita across 83 countries. It involves automated data cleaning, exploratory data analysis, and multiple linear regression modeling.

## Project Structure

- **Data/**: Processed and raw datasets.
  - `consolidated_data.csv`: Cleaned dataset with 83 countries and 20 indicators.
  - `indicator_descriptions.csv`: Mapping of indicator codes to human-readable names.
  - **raw/**: Original World Bank CSV files.
- **src/**:
  - `dataProcessing.py`: Python script for data cleaning and merging.
  - `Project1Script.R`: R script for statistical modeling and visualization.
- **Plots/**: Diagnostic and result plots generated by the R script.
- **ProjectDetails/**: Academic requirements and templates for the project.

## Workflow

### 1. Data Processing (`dataProcessing.py`)
The Python script automates the creation of a cross-sectional dataset:
- **Latest Value Extraction**: Identifies the most recent non-null value for 20 indicators across the 1960–2024 timeframe for each country.
- **Filtering**: Retains only countries with a full set of all 20 indicators to ensure a balanced dataset for modeling.
- **Metadata Merging**: Joins indicator data with regional metadata.

### 2. Statistical Analysis (`Project1Script.R`)
The R script performs exhaustive regression analysis:
- **Exploratory Analysis**: Visualizes correlations between 20 predictors and GNI per capita.
- **Model Testing**: Evaluates 27 different linear and exponential models.
- **Model Selection**: Uses **Best Subset Regression** (via `olsrr`) to compare models based on Adjusted R-squared, AIC, and BIC (SBC).
- **Assumptions Testing**: Validates OLS assumptions using diagnostic plots (Residuals vs. Fitted, Q-Q plots, etc.) and VIF for multicollinearity.

## Final Model

The analysis concluded that a 3-predictor exponential model provides the best balance of parsimony and explanatory power:

**Model Formula:**
`log(GNI_per_Capita) ~ Advanced_Education + Urban_Population + Access_to_Electricity`

### Performance Metrics:
- **Adjusted R-squared**: 0.6979
- **F-statistic**: 64.15 (p-value < 2.2e-16)
- **Key Findings**: All three predictors are highly significant (p < 0.01), and the model satisfies all OLS assumptions (linearity, homoscedasticity, normality of residuals, and independence).

## How to Run

1. **Prepare Data**: Run `python src/dataProcessing.py` (requires `pandas`).
2. **Run Analysis**: Execute `src/Project1Script.R` in RStudio or via Rscript.
   - Requires libraries: `tidyverse`, `ggplot2`, `ggfortify`, `olsrr`, `car`, `performance`, `patchwork`.
